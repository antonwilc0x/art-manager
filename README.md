A command line interface for storing art request, commission, and YCH information. The application is a work in progress but contribution is welcomed.

## Requirements

### Prerequisites

- Rust 1.32+
- Diesel SQLite Cli

It's recommended for Windows users to use the SQLite bundle option: ``cargo install diesel_cli --no-default-features --features "sqlite-bundled"``.

### Platforms

- Linux 2.16.8+
- macOS 10.7+
- Windows 7+
    - Windows 10 October 2018 Update or above recommended
